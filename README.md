# Notion Custom Agent News Curation

Notionカスタムエージェントで、技術領域ごとの最新情報を収集・要約するための指示書を管理するリポジトリ。

各指示書は、公式一次情報だけを情報源とし、Notionデータベースに毎朝の最新情報ページを作成する前提で作成している。単なるニュースの羅列ではなく、何が変わったか、開発時の活用イメージ、導入方法、注意点、公式リンク、関連タグまで含めることで、チームが次の行動に移しやすい形に整える。

## 指示書一覧

- `ai-news.md`: AIサービス・AI開発支援ツール向け
- `frontend-news.md`: フロントエンド技術・Webプラットフォーム・UI開発ツール向け
- `backend-news.md`: バックエンド技術・クラウド基盤・データベース・API開発ツール向け

## 共通方針

- 公式Changelog、Release notes、Blog、Docsなどの一次情報のみを使用
- X、ニュース記事、個人ブログ、まとめ記事、解説記事は使用しない
- 公式リンクが見つからないトピックは採用しない
- 直近2〜3か月のNotion既存ページを確認し、重複を除外
- 3〜10件に絞り、重要な更新だけを掲載
- 「今日の最新情報」と「最近の重要アップデート（再掲）」の2セクション構成
